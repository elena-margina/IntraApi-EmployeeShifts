EF Core migrations: open Package Manager Console:

PM> Add-Migration MigrationName -Context ContextName
(example: PM> Add-Migration InitialMigration -Context BethanysPieShopDbContext
The migration generate two files:
	20241229174903_InitialMigration.cs
	BethanysPieShopDbContextModelSnapshot.cs
)


PM> Remove-Migration -> if you want to remove migration
To create the database and the tables, we must execute this command:
PM> Update-Database

            services.AddDbContext<IntraApiDBContext>(options =>
